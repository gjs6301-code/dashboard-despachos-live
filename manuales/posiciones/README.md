# Workforce Platform - Paquete documental de posiciones

Este paquete contiene plantillas base para crear, entrenar y auditar posiciones que usan Workforce Platform.

Uso recomendado:

1. Seleccionar la posicion que se desea crear o actualizar.
2. Revisar proposito, responsabilidades, pantallas, botones y permisos.
3. Ajustar nombres de responsables, horarios, SLAs y KPIs a la operacion real.
4. Adjuntar screenshots de las pantallas indicadas por el Auditor de Procesos.
5. Usar la matriz RACI para confirmar quien ejecuta, aprueba, consulta e informa.
6. Publicar la version aprobada como manual operativo oficial.

Documentos incluidos:

- `gerente-operaciones.md`
- `auditor-procesos.md`
- `encargado-operativo.md`
- `auxiliar-almacen-empaque.md`

Regla de gobierno:

Las recomendaciones del Auditor de Procesos pueden aprobarse o rechazarse en la plataforma, pero ningun cambio de codigo se ejecuta automaticamente. La implementacion requiere una instruccion explicita a Codex.
